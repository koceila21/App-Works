# 🚀 WorksUrgent

Plateforme de jobs urgents avec géolocalisation et notifications temps réel.

## Fonctionnalités

- Auth Supabase
- Jobs internes / externes
- Carte interactive
- Notifications temps réel
- Chat simple
- page de publication de frorums de recrutement 
## Setup

Remplacer dans app.js :

SUPABASE_URL
SUPABASE_KEY

## Déploiement

- GitHub
- Vercel
